# Screenshot Guide for SmartChat iOS App

This guide helps you capture screenshots highlighting all core features of the SmartChat app.

## üì∏ How to Take Screenshots

### On iOS Simulator:
1. **Command + S** - Takes a screenshot and saves to Desktop
2. **File ‚Üí Save Screen** - Alternative method
3. Screenshots are saved as PNG files with timestamp

### On Physical iOS Device:
1. **Power + Volume Up** (iPhone X and later)
2. **Home + Power** (iPhone 8 and earlier)
3. Screenshots saved to Photos app

## üé® Screenshot Best Practices

### Quality Tips:
1. **Use iOS Simulator** for consistent, high-quality screenshots
2. **Remove debug banner** (already done in your app)
3. **Use realistic data** - populate with sample conversations
4. **Consistent styling** - use same theme for related screenshots
5. **Clean UI** - ensure no test data or placeholder text

### Organization:
- Name files descriptively: `01-auth-signup.png`, `02-chat-list.png`
- Create folders: `screenshots/ios/`, `screenshots/android/`
- Keep original resolution (don't compress until final export)

### Recommended Screenshot Sizes:
- **App Store**: 6.5" display (iPhone 14 Pro Max) - 1284 x 2778 pixels
- **Marketing**: Use highest resolution available
- **Documentation**: 2x resolution for clarity
